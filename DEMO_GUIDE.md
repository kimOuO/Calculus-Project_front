# 快速演示指南

## 🎯 核心功能演示

### 1. 查看學生成績（動態路由）

**路徑**: 學生管理 → 點擊"成績"鏈接 → 學生成績詳情

```
訪問: http://localhost:3000/students
操作:
1. 在學生列表中找到任意學生
2. 點擊該學生行中的"成績"鏈接
3. 自動跳轉到: /students/[學號]/scores
4. 查看該學生的所有考試成績
```

**特點**:
- ✅ 使用學號作為路由參數（不是 UUID）
- ✅ 顯示所有考試成績和總分
- ✅ 支持實時更新成績

---

### 2. 更新學生成績

**路徑**: 學生成績詳情頁面 → 更新成績按鈕

```
在學生成績頁面 (/students/[學號]/scores):
1. 點擊"更新成績"按鈕
2. 選擇要更新的考試項目（小考1/期中考/小考2/期末考）
3. 輸入分數（0-100）
4. 點擊"更新"
5. 成績即時刷新
```

---

### 3. 成績總覽與管理

**路徑**: http://localhost:3000/scores

**功能**:
1. **查看全班成績**
   - 顯示所有學生的完整成績
   - 支持按學期篩選
   - 點擊任意學生行跳轉到其成績詳情

2. **計算期末總成績**
   ```
   步驟:
   1. 輸入學期（如: 1141）
   2. 輸入及格分數（如: 60）
   3. 點擊"計算期末總成績"
   4. 系統自動計算全班總分並更新學生狀態
   ```

3. **查看考試統計**
   ```
   步驟:
   1. 選擇學期
   2. 選擇考試項目（小考1/期中考等）
   3. 點擊"查看統計"
   4. 彈出統計信息（平均分、中位數、總人數）
   ```

---

### 4. 考試管理

**路徑**: http://localhost:3000/tests

**功能**:
1. **創建考試**
   ```
   1. 點擊"新增考試"
   2. 填寫表單:
      - 考試名稱: 第一次小考
      - 考試日期: 2024-03-15
      - 考試範圍: Chapter 1-3
      - 學年學期: 1141
      - 權重: 0.2 (可選)
   3. 提交
   ```

2. **設定權重**
   ```
   前提: 該學期已有考試記錄
   步驟:
   1. 選擇學期（篩選器）
   2. 點擊"設定權重"
   3. 為每場考試輸入權重（確保總和為 1.0）
   4. 系統實時顯示權重總和
   5. 提交
   ```

---

### 5. 學生管理

**路徑**: http://localhost:3000/students

**功能**:
1. **新增學生**
2. **查看學生列表**
3. **篩選學生**（按學期）
4. **操作**:
   - 成績: 跳轉到成績詳情
   - 編輯: 待實現
   - 刪除: 帶確認提示（級聯刪除成績）

---

## 🔄 完整工作流示例

### 場景: 新學期開始

```
第一步: 創建考試
訪問: /tests
操作: 創建 4 場考試（小考1、期中考、小考2、期末考）

第二步: 添加學生
訪問: /students
操作: 批量添加學生

第三步: 錄入成績
方式 A: 單個學生
  1. 在學生列表點擊"成績"
  2. 進入該學生成績頁面
  3. 更新各項考試成績

方式 B: 成績總覽
  1. 訪問 /scores
  2. 點擊任意學生
  3. 更新成績

第四步: 設定權重
訪問: /tests
操作: 設定權重（總和 1.0）

第五步: 計算總成績
訪問: /scores
操作: 
  1. 輸入學期和及格分數
  2. 點擊"計算期末總成績"
  3. 系統自動計算並更新學生狀態

第六步: 查看統計
訪問: /scores
操作:
  1. 選擇考試項目
  2. 查看統計數據
```

## 🎨 視覺效果說明

### 成績顏色編碼
- **綠色** (≥80): 優秀
- **藍色** (60-79): 及格
- **紅色** (<60): 不及格
- **灰色** (空值): 未輸入

### 狀態標籤
- 🟢 **修業中**: 綠色標籤
- 🔵 **修業完畢**: 藍色標籤
- 🔴 **被當**: 紅色標籤
- ⚫ **二退**: 灰色標籤

---

## ⚠️ 注意事項

1. **權重設定必須在計算總成績前完成**
2. **刪除學生會同時刪除其所有成績**
3. **計算總成績需要學生有完整的四項成績**
4. **二退學生不參與成績計算**

## 🐛 測試建議

### 基本測試
1. ✅ 創建學生
2. ✅ 通過學號訪問成績頁面
3. ✅ 更新成績
4. ✅ 查看成績總覽
5. ✅ 創建考試
6. ✅ 設定權重
7. ✅ 計算總成績

### 邊界測試
- 輸入超出範圍的分數（應該被攔截）
- 權重總和不為 1.0（應該被攔截）
- 刪除有成績的學生（應該級聯刪除）
- 計算沒有權重的學期總成績（應該失敗）
